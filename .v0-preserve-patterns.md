# V0 Preservation Patterns

This file documents patterns that should be preserved when V0 regenerates code.

## Critical Patterns to Preserve

### 1. Route Handler Pattern
**Location:** `app/[view]/page.tsx` (spread-builder, guidebook, reading, gallery)

**Pattern:**
```typescript
// Re-export the main app - it will handle routing based on URL
export { default } from "../page";
```

**Why:** Maintains unified SPA architecture while allowing direct URL access.

---

### 2. URL Synchronization
**Location:** `app/page.tsx`

**Pattern:**
```typescript
const PATH_TO_VIEW: Record<string, View> = {
  "/": "home",
  "/gallery": "gallery",
  "/reading": "reading",
  "/guidebook": "guidebook",
  "/spread-builder": "spread-builder",
};

// Sync view with URL on mount and when URL changes
useEffect(() => {
  if (typeof window === "undefined") return;
  const updateViewFromPath = () => {
    const path = window.location.pathname;
    const viewFromPath = PATH_TO_VIEW[path] || "home";
    setCurrentView(viewFromPath);
  };
  updateViewFromPath();
  window.addEventListener("popstate", updateViewFromPath);
  return () => window.removeEventListener("popstate", updateViewFromPath);
}, []);
```

**Why:** Enables bookmarkable URLs and browser navigation.

---

### 3. localStorage SSR Safety
**Location:** Any client component using localStorage

**Pattern:**
```typescript
useEffect(() => {
  if (typeof window === "undefined") return;
  // ... localStorage code
}, []);
```

**Why:** Prevents SSR hydration errors.

### 4. AI Model Provider Imports
**Location:** `app/api/ai-reading/route.ts`, `app/api/daily-wisdom/route.ts`

**Pattern:**
```typescript
import { anthropic } from "@ai-sdk/anthropic";
import { openai } from "@ai-sdk/openai";
import { google } from "@ai-sdk/google";
import { groq } from "@ai-sdk/groq";
import { xai } from "@ai-sdk/xai";

const model = anthropic("claude-sonnet-4-20250514");
```

**Why:** Required for proper AI SDK functionality.

---

### 5. Navigation URL Update
**Location:** `app/page.tsx` - `handleNavigate` function

**Pattern:**
```typescript
const handleNavigate = (view: View) => {
  setCurrentView(view);
  setActiveSpread(null);
  const path = view === "home" ? "/" : `/${view}`;
  if (typeof window !== "undefined" && window.location.pathname !== path) {
    router.push(path, { scroll: false });
  }
  if (typeof window !== "undefined") {
    window.scrollTo({ top: 0, behavior: "smooth" });
  }
};
```

**Why:** Keeps URL in sync with view state.
